# Movie Analytics Pipeline

## Overview
This project transforms raw movie watch logs from iPhone Notes into a structured dataset using Python and Pandas. It performs exploratory data analysis to uncover viewing patterns, ratings, and trends over time.

## Features
- Text parsing using regex
- Data cleaning and structuring with Pandas
- Feature engineering (year, rolling averages)
- Exploratory data analysis
- Trend visualization

## Technologies Used
- Python
- Pandas
- Regex
- Jupyter Notebook
- Matplotlib

## Key Insights
- Average rating trends over time
- Most watched and highest rated movies
- Viewing frequency by year
- Smoothed rating behavior using rolling averages
